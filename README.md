# Amazon Prime TV Shows & Movies – Exploratory Data Analysis
## Project Overview

This project performs Exploratory Data Analysis (EDA) on an Amazon Prime Movies and TV Shows dataset. The goal is to extract meaningful insights about content distribution, ratings, genres, release trends, and other key factors that can help understand platform content strategy and audience preferences.

The analysis is done using Python in Google Colab Notebook.

## Objectives

Understand the distribution of Movies vs TV Shows

Analyze content trends over the years

Study IMDb ratings and vote patterns

Identify popular genres

Examine runtime distribution

Generate business insights based on data findings

## Dataset Information

The dataset contains information such as:

Title

Type (Movie / TV Show)

Release Year

IMDb Score

IMDb Votes

Runtime

Genre

Country

## Key Insights

Majority of content consists of movies compared to TV shows

Most content is concentrated in recent years

Movies with runtime between 80–120 minutes is the sweet spot for runtime

Certain genres dominate the platform

Highly rated titles do not always have the highest vote counts

## Business Recommendations

Increase content diversity from multiple countries

Focus on high-performing genres like SciFi, War and Fantasy

Maintain optimal movie runtime range

Add more high-quality TV shows
